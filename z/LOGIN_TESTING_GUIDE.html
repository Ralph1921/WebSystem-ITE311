🚨 Common Issues & Solutions

If Dashboard Still Doesn't Show:

  1. Clear Browser Cache: Ctrl+F5 or Ctrl+Shift+R
  2. Check Session Files: Look in writable/session/ folder
  3. Restart XAMPP: Stop and start Apache service
  4. Check Permissions: Ensure writable/ folder has write permissions

If You See "Welcome terrado@gmail.com" but Page is Blank:

  • This indicates the session is working but the view is not rendering
  • Check browser developer tools (F12) for JavaScript errors
  • Verify all CSS/Bootstrap files are loading correctly
