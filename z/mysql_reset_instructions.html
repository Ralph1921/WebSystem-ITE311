Method 1: Using XAMPP Control Panel

Step 1: Stop MySQL

  1. Open XAMPP Control Panel as Administrator
  2. Click "Stop" next to MySQL
  3. Wait until it shows "Stopped"

Step 2: Edit MySQL Configuration

  1. Open: C:\xampp\mysql\bin\my.ini in Notepad (as Administrator)
  2. Find the [mysqld] section
  3. Add this line: skip-grant-tables
  4. Save the file

Step 3: Start MySQL

  1. In XAMPP Control Panel, click "Start" next to MySQL
  2. Wait until it shows "Running"

Step 4: Reset Password

  1. Open Command Prompt as Administrator
  2. Run these commands one by one:
cd C:\xampp\mysql\bin
mysql.exe -u root

In the MySQL prompt, run:

USE mysql;
UPDATE user SET authentication_string = '', plugin = 'mysql_native_password' WHERE User = 'root';
FLUSH PRIVILEGES;
EXIT;

Step 5: Remove skip-grant-tables

  1. Open C:\xampp\mysql\bin\my.ini again
  2. Remove the line skip-grant-tables
  3. Save the file

Step 6: Restart MySQL

  1. In XAMPP Control Panel, click "Stop" then "Start" for MySQL
